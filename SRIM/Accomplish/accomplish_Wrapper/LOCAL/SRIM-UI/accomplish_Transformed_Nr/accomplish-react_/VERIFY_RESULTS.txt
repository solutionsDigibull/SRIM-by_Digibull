HOW TO LAUNCH
  1. Double-click  LAUNCH-WEB.bat  (in this folder).
  2. It opens two windows: "Accomplish DAEMON :9234" and "Accomplish WEB :5173",
     then opens http://localhost:5173 in your browser.
  3. If the DAEMON window shows a better-sqlite3 / NODE_MODULE_VERSION error,
     you are on the wrong Node. Run:  nvm use 24  then delete node_modules and
     re-run the .bat. (Project requires Node 24+.)
