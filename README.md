# DigiBull-SRIM (Accomplish)

This repository is the full **DigiBull / SRIM** working tree for the **Accomplish** desktop AI co-worker, mirrored exactly as it sits under `C:\Digibull`.

The **live, current build** (maintained by **Nagabhushana Raju S**) is the **`Nr`** project:

```
SRIM/Accomplish/accomplish_Wrapper/LOCAL/SRIM-UI/accomplish_Transformed_Nr/accomplish-react_
```

It is a **pnpm monorepo** (Node 24 + pnpm 10.33.0):

| App | Port | Description |
|-----|------|-------------|
| `@accomplish/web`    | http://localhost:5173    | React + Vite web UI |
| `@accomplish/daemon` | http://127.0.0.1:9234    | Background daemon / API |
| `@accomplish/desktop`| -                        | Electron desktop app (Windows) |

---

## Quick start

### Option A - Windows (recommended): one-click installer

From the repo root, double-click **`INSTALL.bat`** (or run it in a terminal). It will:

1. Install **Node.js 24 LTS** (via `winget`) if missing,
2. Activate **pnpm 10.33.0** (via `corepack`),
3. Run `pnpm install` in the live `Nr` project,
4. Optionally launch the app (daemon :9234 + web :5173 + browser) via `LAUNCH-WEB.bat`.

> Dev login: button **"Sign in as Tester"**, or token `srim-dev-token` (user `digibull` / pass `srim-test-2026`).

### Option B - Docker (any OS with Docker)

From the repo root:

```bash
docker compose up --build
```

Then open <http://localhost:5173>. This runs the **headless web build** (daemon + Vite).
The Electron *desktop* GUI is not run in Docker - use Option A for the desktop app.

## Repository notes

- **`node_modules/` is not committed** - it is regenerated by `INSTALL.bat` / Docker.
- Large binaries (`*.zip`, `*.exe`) are stored via **Git LFS**. Install [git-lfs](https://git-lfs.com)
  and run `git lfs pull` after cloning if you need them.
- The tree also contains the earlier `ELECTRON` and `Chaithanya` snapshots for reference;
  `Nr` is the authoritative current version.
